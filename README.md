# Playground Repository - Code Examples

This repository showcases Python and SQL implementations for interview demonstrations, featuring data processing, ETL pipelines, advanced SQL analytics, and a simple AI agent.

## Repository Contents

### Python & SQL Examples (`python_sql_examples/`)
- **Advanced Data Processing**: CSV loading with type conversion, data validation, monthly aggregation using built-in modules
- **Complex SQL Scripts**: User analytics with views, joins, aggregations, and sample data insertion

### Data Pipeline (`data_pipeline/`)
- **Simple ETL Pipeline**: Extracts sales data from CSV, calculates totals, loads into SQLite database, generates summary report
- **Clean Architecture**: Modular functions for each ETL step
- **Business Impact**: Automates manual reporting process

### Complex SQL Query (`complex_sql/`)
- **Customer Lifetime Value Analysis**: Multi-CTE query with cohort segmentation, retention curves, and advanced window functions
- **Analytics Depth**: Combines ranking, segmentation, and predictive metrics
- **Detailed Explanation**: Comprehensive breakdown of SQL techniques and business rationale

### AI Agent Application (`ai_agent/`)
- **Improved Conversational Agent**: Keyword-based intent classification with conversation memory
- **Context Awareness**: Remembers recent topics and provides relevant follow-up responses
- **Class-Based Design**: Uses object-oriented structure for state management
- **Demo Prompts**: Sample questions showing context awareness and intent classification
- **Walkthrough**: Step-by-step guide with example conversation flow

## Technical Highlights
- **No External Dependencies**: All code uses Python standard library only
- **Interview-Ready**: Complex enough to demonstrate skills, simple enough to explain clearly
- **Real Artifacts**: Actual runnable code with business context

## Running the Code
- Python scripts: `python folder/script.py`
- SQL: Execute in your preferred database (SQLite recommended for demos)
- AI Agent: `python ai_agent/agent.py` for interactive demo

This repository demonstrates data engineering and AI development skills.
